Choose a short feature name, save the current plan to `./PLAN-{feature-name}.md`
and start implementing the plan step by step.
Keep the plan file up to date with the progress.
Stop at every step/phase completion to ask the user for validation and permission to implement the next step.
Finally, ask the user if everything is fine and, if the answer is positive, delete `./PLAN-{feature-name}.md`
